# How to use

## Installation

```
yarn add @xchainjs/xchain-util
```

The following peer dependencies have to be installed into your project. These are not included in `@xchainjs/xchain-util`.

```
yarn add bignumber.js
```

## Development

### Build

```bash
yarn build
```

### Tests

```bash
yarn test
```
